# 天氣預報頁面

## 使用技術

- Vue.js 框架開發專案。
- Vue CLI 3。
- Bootstrap 與 Vue-Bootstrap 套件刻畫頁面。
- axios 串接中央氣象局 open data 資料。
- Vue ApexChart 套件圖表使用。
- moment.js 時間套件使用。
- 攝氏轉換華氏計算。
